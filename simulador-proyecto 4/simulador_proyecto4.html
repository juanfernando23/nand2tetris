Simulador de Mult.asm

Este simulador muestra el proceso de multiplicación por sumas repetidas.

R0: 5
R1: 3
R2 (resultado): 0
i (contador): 3 
// Bucle: repetir r1 veces
while i > 0:
    // Sumar r0 al resultado
    r2 = r2 + r0
    // Decrementar el contador
    i = i - 1
